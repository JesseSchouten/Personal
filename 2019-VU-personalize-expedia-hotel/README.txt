This repository contains the models for the Kaggle Personalize Expedia Hotel Searches challenge, which was part of the Data Mining Techniques course, taught at Vrije Universiteit Amsterdam, Amsterdam from the 1-4-2019 - 27-5-2019.

For a description of the competition: https://www.kaggle.com/c/vu-dmt-2assignment/overview, the competition is inspired from the 2013 Personalize Expedia Hotel Searches challenge (https://www.kaggle.com/c/expedia-personalized-sort).

The file main_models.ipynb and main_models.py contain the models, feature engineering, etc that was used to create the outcome. Note that this file is raw, and one might have to do effort to get it to run. The dataset is large and can be found at https://www.kaggle.com/c/vu-dmt-2assignment/data, or similar data can be found at https://www.kaggle.com/c/expedia-personalized-sort/data. 

Kaggle placement: 11/151 (https://www.kaggle.com/c/vu-dmt-2assignment/leaderboard)

Methods used: XGBoost, LambdaMART, Random Forest.
