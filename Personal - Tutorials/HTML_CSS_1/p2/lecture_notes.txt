Note 1:
3 methods for adding CSS:
1 - Inline CSS, add directly to HTML file. DONT DO THIS.
2 - Internal CSS, Using <style> tags within a document. Not best option: fattens up HTML file, is also not reusable.
3 - External CSS, Linking external CSS file.

Note 2:
CSS selectors:
a {background-color:yellow;}
selector - declaration start - property - propert/value seperator - value - declaration separator - declaration end.

Note 3:

alternative - rgb(0,0,255) (red green blue)
